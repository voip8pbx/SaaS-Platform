## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

1.  **Clone the Repository**
    ```bash
    git clone https://github.com/yourusername/travel-panner.git
    cd travel-panner
    ```

2.  **Setup SuperAdmin Controller**
    ```bash
    cd SuperAdmin
    npm install
    npm run dev:all
    # Runs on http://localhost:3000
    ```

3.  **Setup Client Template**
    Open a new terminal:
    ```bash
    cd ../Travelpannerwebsite
    npm install
    npm run dev:all
    # Runs on http://localhost:3001
    ```

### Usage
1.  Open **SuperAdmin** at `http://localhost:3000`.
2.  Change the **"Layout Style"** to *Classic* or *Modern*.
3.  Click **"Create Website"** or **"Save Changes"**.
4.  Refresh **Client Website** at `http://localhost:3001` to see the complete design transformation!
